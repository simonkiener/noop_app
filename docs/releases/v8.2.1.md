# NOOP v8.2.1 Release Notes

Released: July 10, 2026

## What's New

### Design & Customization
- **Classic Gauge Charts:** Reverted the liquid bubble progress indicators back to the clean, classic circular progress rings (Gauge Charts) for all metrics (Sleep, Recovery, Strain) across iOS, macOS, and Android.
- **Unified Naming:** Renamed the core health metrics across the app interfaces to better align with the standard terms:
  - **Charge** is now **Recovery**
  - **Rest** is now **Sleep**
  - **Effort** is now **Strain**
- **Reordered Today Dashboard:** Rearranged the hero columns on the main Today dashboard on both iOS/macOS and Android to show **Sleep**, then **Recovery**, and then **Strain**.

### Bug Fixes
- **Effort Scale Preference:** Fixed a bug where the Effort (Strain) hero cell in the new v8 Liquid Today dashboard would ignore the user-selected scale (0-100 or 0-21) and always display on the 0-100 scale. The dashboard now properly respects the chosen scale and decimal format when displaying Strain.
